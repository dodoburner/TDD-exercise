# TDD-exercise

TDD-exercise is a project that contains a class `solver.rb` that have methods solve factorial, solve fibonacci and solve reverse. And a test file `solver_spec.rb` that contains the tests for the methods.

## Built With

- Ruby
- Rspec

## Getting Started
